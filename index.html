Proceso digital de imágenes

Instrucciones

(1) Escoger la imagen: Utilizar el botón "Seleccionar Archivo" para escoger la imagen.
(2) Aplicar el filtro: Una vez que haya cargado la imagen deseada en la sección 3, presionar uno de los botones para aplicar los filtros. Es necesario presionar el botón Cargar o Limpiar Filtro antes de volver a aplicar otro filtro o, de lo contrario, se aplicarán múltiples filtros a la misma imagen. Para el filtro mosaico es necesario especificar el tamaño con el que se trabajará antes de presionar el respectivo botón.
(3) Observar el resultado: Aquí se mostrará la imagen cuando se cargue, cuando se le apliquen filtros, etc. Además, aquí se indicará cualquier error que surja durante la ejecución del programa.
(4) Descarga tu imagen: Utiliza el botón Descargar para descargar tu imagen.
